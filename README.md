# AranApp

یک سوپراپ ماولار مبتنی بر وب۳ و بلاکچین با الکترون

## ویگیهای اصلی
- معماری ماولار کامل
- سیستم Event Bus برای ارتباط بین ماولها
- 8 اکاردئون برای دسترسی سریع به ابزارها
- سیستم تبهای پیشرفته
- پشتیبانی از تم روشن و تاریک
- بروزر دسکتاپی با الکترون
